# 我的个人博客

这是使用 **Jekyll** 搭建的个人博客，托管在 **GitHub Pages** 上。

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/fyl34222220/fyl34222220.github.io.git

# 新建文章
# 在 _posts 目录下创建文件，格式：YYYY-MM-DD-标题.md

# 提交并推送
git add .
git commit -m "新文章"
git push origin main
```

## 📝 写文章

在 `_posts/` 目录下创建 Markdown 文件：

```markdown
---
layout: post
title: "文章标题"
date: YYYY-MM-DD HH:MM:SS +0800
categories: 分类
tags: [标签1, 标签2]
---

正文内容...
```

## 📄 项目结构

```
├── _config.yml      # 站点配置
├── _posts/          # 博客文章
├── index.html       # 首页
└── README.md        # 说明文档
```
